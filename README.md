# NailBook Pro - API Documentation & QA Guide

Цей проект розділений на бекенд (Express/Prisma/PostgreSQL) та фронтенд (React Native Expo).

## Інструкція для Тестувальників (QA Guide)

### Як зібрати APK
Проект налаштований для збірки Android APK через Expo Application Services (EAS).
1. Встановіть EAS CLI: `npm install -g eas-cli`
2. Авторизуйтеся у вашому акаунті Expo: `eas login`
3. Зберіть проект для Android: 
   ```bash
   cd frontend
   eas build -p android --profile production
   ```
4. Після завершення збірки ви отримаєте посилання на завантаження `.apk` файлу, який можна встановити на пристрій або емулятор.

### Основні Сценарії Тестування
1. **Реєстрація та Ролі:** Зареєструйтесь як Майстер. Встановіть робочий час (наприклад з 9 до 18).
2. **Клієнтський шлях:** Зареєструйтесь як Клієнт. Запишіться до Майстра на вільний слот.
3. **Чат (Socket.io):** Відкрийте чат між Клієнтом та Майстром. Перевірте миттєву доставку повідомлень та зміну статусу на "прочитано".
4. **Галерея та Оплата:** Майстер публікує роботу. Клієнт ставить лайк/додає в обране. Майстер генерує QR код для оплати, клієнт його бачить.
5. **Статистика (Спринт 5):** Перевірте, чи змінюються графіки "Статистики" у профілі Майстра після створення нових записів.
6. **Telegram та Push:** Отримайте лінк для телеграм бота (у налаштуваннях профілю) та перейдіть у бот. Натисніть "Start", має прийти вітальне повідомлення.

## Документація API
Host: `http://localhost:3000` (або домен на Railway)

| Endpoint | Method | Role | Опис |
|----------|--------|------|------|
| `/api/auth/register` | POST | 🌍 | Реєстрація користувача (вказати `role`) |
| `/api/auth/login` | POST | 🌍 | Авторизація (повертає JWT токен) |
| `/api/master/settings` | PUT | 💅 | Зберегти графік майстра |
| `/api/master/slots` | GET | 💅 | Отримати згенеровані вікна |
| `/api/client/appointments` | POST | 👤 | Створити запис до майстра |
| `/api/chats` | GET | 🌍 | Отримати список активних чатів |
| `/api/chats/:id/messages` | GET | 🌍 | Отримати історію повідомлень |
| `/api/gallery` | GET, POST | 🌍/💅 | Перегляд (усі) / Публікація робіт (майстер) |
| `/api/favorites/:id` | POST | 🌍 | Додати/прибрати з обраного |
| `/api/master/payment/setup`| PUT | 💅 | Налаштувати платіжні дані (генерує QR) |
| `/api/telegram/link` | POST | 🌍 | Отримати унікальне посилання на бота |
| `/api/calendar/auth` | GET | 💅 | Отримати лінк для підключення Google Calendar |
| `/api/master/statistics/overview`| GET | 💅 | Основні метрики (доходи, клієнти) |
| `/api/master/statistics/export/pdf`| GET | 💅 | Завантажити PDF-звіт |

*Всі захищені маршрути вимагають заголовок `Authorization: Bearer <token>`*
